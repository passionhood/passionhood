## Selected Work

### Investor Persona Clustering App – Unsupervised Machine Learning  
https://github.com/passionhood/Hood-Data-Science-Portfolio/tree/main/MLUnsupervisedApp

This project is an interactive Streamlit application that uses unsupervised machine learning to identify investor behavior profiles based on portfolio characteristics and trading patterns.

The app allows users to upload data, select features, and experiment with clustering parameters to explore how investor segments emerge without predefined labels.

Key components include:
- K-Means clustering with adjustable number of clusters and initialization methods  
- Hierarchical clustering with dendrogram visualization  
- Principal Component Analysis (PCA) for dimensionality reduction and visualization  
- Model evaluation using silhouette score and elbow method  
- Automated interpretation of clusters into investor personas  
- Interactive interface with downloadable outputs  

This project demonstrates unsupervised learning, dimensionality reduction, and the ability to translate model output into meaningful, real-world interpretations.

### Investment Risk Analyzer – Supervised Machine Learning  
https://github.com/passionhood/Hood-Data-Science-Portfolio/tree/main/MLStreamlitApp

This project is a Streamlit-based application that analyzes financial market data and predicts investment risk using supervised machine learning models.

Users can explore historical market data, engineer features, and train models to classify investment risk while evaluating performance in real time.

Key components include:
- Multiple models including Logistic Regression, Decision Tree, and K-Nearest Neighbors  
- Feature engineering using returns, volatility, and rolling metrics  
- Hyperparameter tuning and model comparison  
- Evaluation using accuracy, precision, recall, F1 score, and ROC-AUC  
- Interactive dashboards for data exploration and visualization  

This project demonstrates predictive modeling, model evaluation, and the ability to build production-ready analytical tools.
